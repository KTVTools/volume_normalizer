# volume_normalizer

這個程式是搭配 Crazy KTV 點歌程式, 調整適當音量, 讓每首歌曲在播放時,
音量都根據 replaygain 的 -89dB 播放, 音量就不會有的歌曲太大聲, 有的太小聲.

本程式為 KTV Tools 中最後一階段的工具程式. 在此程式之前,
需要利用 [Vocal Channel Analyzer](https://github.com/KTVTools/Vocal-Channel-Analyzer)
